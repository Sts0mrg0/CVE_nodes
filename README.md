# CVE_nodes
## 总体目标

完成互联网CVE漏洞的知识整理，普通CVE包含以下内容：

- 漏洞编号
- 漏洞名称
- 漏洞详情
- 漏洞检测代码
- 漏洞利用代码
- 代码使用说明
- 漏洞修复
